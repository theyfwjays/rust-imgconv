# 06. Rotation

Test 90°, 180°, and 270° clockwise rotation.

## Commands

```bash
imgconv TestImg.jpg --to png --rotate 90 -o test_output/06_rotate
imgconv TestImg.jpg --to png --rotate 180 -o test_output/06_rotate
imgconv TestImg.jpg --to png --rotate 270 -o test_output/06_rotate
```

## Output Files

| File | Description |
|------|-------------|
| TestImg_90.png | 90° clockwise rotation |
| TestImg_180.png | 180° rotation |
| TestImg_270.png | 270° clockwise rotation |
