# 02. Multi-Format Conversion

Convert a single input file to multiple formats simultaneously.

## Commands

```bash
imgconv TestImg.jpg --to png,bmp,gif -o test_output/02_multi_format
```

## Output Files

| File | Format |
|------|--------|
| TestImg.png | PNG |
| TestImg.bmp | BMP |
| TestImg.gif | GIF |
