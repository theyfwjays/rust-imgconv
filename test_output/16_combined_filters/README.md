# 16. Combined Filters

Apply multiple filters and options simultaneously in a single pipeline.

## Commands

```bash
imgconv TestImg.jpg --to png --grayscale --brightness 30 --contrast 10.0 --width 500 -o test_output/16_combined_filters
```

## Output Files

| File | Description |
|------|-------------|
| TestImg.png | Grayscale + brightness + contrast + resize combined |
