# 13. Brightness Adjustment

Adjust brightness. Positive = brighter, negative = darker.

## Commands

```bash
imgconv TestImg.jpg --to png --brightness 50 -o test_output/13_brightness    # Brighter
imgconv TestImg.jpg --to png --brightness -50 -o test_output/13_brightness   # Darker
```

## Output Files

| File | Description |
|------|-------------|
| TestImg_bright50.png | Brightness +50 |
| TestImg_dark50.png | Brightness -50 |
