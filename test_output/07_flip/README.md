# 07. Flip

Test horizontal and vertical flipping.

## Commands

```bash
imgconv TestImg.jpg --to png --flip horizontal -o test_output/07_flip
imgconv TestImg.jpg --to png --flip vertical -o test_output/07_flip
```

## Output Files

| File | Description |
|------|-------------|
| TestImg_horizontal.png | Horizontal flip (mirror) |
| TestImg_vertical.png | Vertical flip (upside down) |
